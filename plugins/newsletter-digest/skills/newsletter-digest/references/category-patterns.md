# Newsletter Category Patterns

This file defines rules for auto-categorizing newsletters by topic. Rules are evaluated top-to-bottom within each category.

## AI Tools & Workflows

**From patterns:**
- `lenny+how-i-ai@substack.com`
- `lenny@substack.com` (when subject contains AI tool names)

**Subject patterns:**
- Claude, Cursor, Codex, MCP, Opus, Haiku, Sonnet
- vibe cod, agent, automation, agentic
- v0, Lovable, Bolt, Replit
- coding tools, dev tools, IDE

**Category description:** AI coding tools, development workflows, agent-based automation, no-code/low-code platforms, practical AI implementation
