# Commercial Real Estate Glossary

## Property & Transaction Terms

| Term | Definition |
|------|------------|
| **Absorption** | Net change in occupied space over a period. Positive = more space leased than vacated. |
| **BOV (Broker Opinion of Value)** | Broker's estimate of property value, less formal than appraisal. |
| **Cap Rate (Capitalization Rate)** | NOI / Purchase Price. Measures investment return. Lower = more expensive/less risky. |
| **Class A/B/C** | Property quality tiers. A = newest/best, C = older/lower quality. |
| **Comps (Comparables)** | Similar properties used to value/price a subject property. |
| **FSG (Full Service Gross)** | Lease type where landlord pays all operating expenses. |
| **LOI (Letter of Intent)** | Non-binding proposal outlining deal terms. |
| **Net Lease (NNN/Triple Net)** | Tenant pays base rent + taxes, insurance, maintenance. |
| **NOI (Net Operating Income)** | Revenue minus operating expenses (before debt service). |
| **PSF (Per Square Foot)** | Standard unit for pricing ($25/SF/year rent, $200/SF sale price). |
| **RSF (Rentable Square Feet)** | Space tenant pays for (includes common area allocation). |
| **TI (Tenant Improvements)** | Buildout allowance landlord provides to tenant. |
| **USF (Usable Square Feet)** | Space tenant actually occupies (excludes common areas). |

## Lease Terms

| Term | Definition |
|------|------------|
| **Base Rent** | Fixed rental amount before additional charges. |
| **CAM (Common Area Maintenance)** | Tenant's share of building operating costs. |
| **Escalation** | Annual rent increase (fixed % or CPI-linked). |
| **Free Rent / Abatement** | Period where tenant doesn't pay rent (concession). |
| **Gross Lease** | Landlord pays most/all operating expenses. |
| **Lease Expiration** | When current lease ends (prospecting trigger!). |
| **Option to Renew** | Tenant's right to extend lease at predetermined terms. |
| **Term** | Length of lease (typically 3-10 years for commercial). |

## Market & Investment Terms

| Term | Definition |
|------|------------|
| **1031 Exchange** | Tax-deferred swap of investment properties. |
| **Cash-on-Cash Return** | Annual cash flow / initial cash investment. |
| **Core/Core+/Value-Add/Opportunistic** | Investment risk/return profiles (low to high). |
| **Debt Service** | Mortgage payments (principal + interest). |
| **DSCR (Debt Service Coverage Ratio)** | NOI / Debt Service. Lenders want >1.25x typically. |
| **IRR (Internal Rate of Return)** | Annualized return accounting for time value of money. |
| **LTV (Loan-to-Value)** | Loan amount / property value. Lenders cap at 65-75% typically. |
| **Pro Forma** | Projected financial performance of property. |
| **Stabilized** | Property at normal occupancy and operating performance. |
| **Submarket** | Geographic subdivision of metro area for analysis. |
| **Vacancy Rate** | Percentage of space unoccupied. |

## Broker & Industry Terms

| Term | Definition |
|------|------------|
| **CCIM** | Certified Commercial Investment Member - prestigious CRE designation. |
| **Co-Broker** | Broker on other side of transaction (splits commission). |
| **Exclusive Listing** | Broker has sole right to market property. |
| **Listing Broker** | Represents seller/landlord. |
| **Open Listing** | Multiple brokers can market property. |
| **Pocket Listing** | Off-market property not publicly advertised. |
| **Principal Broker** | Licensed broker who oversees firm's agents. |
| **Procuring Cause** | Broker whose efforts resulted in transaction (commission entitlement). |
| **SIOR** | Society of Industrial and Office Realtors - designation for top producers. |
| **Tenant Rep** | Broker representing tenant (vs. landlord). |

## Technology & Data Platforms

| Platform | Purpose |
|----------|---------|
| **CoStar** | Dominant CRE data provider (property info, comps, analytics) |
| **LoopNet** | Public CRE listing marketplace (owned by CoStar) |
| **Reonomy** | Alternative CRE data provider (property ownership focus) |
| **REIS** | CRE market research and analytics (owned by Moody's) |
| **Yardi** | Property management software platform |
| **RealPage** | Multifamily management and data platform |
| **Buildout** | CRE marketing/proposal software |
| **ClientLook** | CRE-specific CRM |
| **AscendixRE** | CRE-specific CRM (Salesforce-based) |

## Deal Structure Terms

| Term | Definition |
|------|------------|
| **As-Is** | Property sold in current condition, no repairs. |
| **Due Diligence** | Investigation period after contract signing. |
| **Earnest Money** | Deposit showing buyer's serious intent. |
| **Escrow** | Third party holds funds until closing. |
| **Phase I ESA** | Environmental Site Assessment (pollution risk check). |
| **Survey** | Legal measurement of property boundaries. |
| **Title Insurance** | Protects against ownership disputes. |
| **Zoning** | Government designation of allowed property uses. |

## Industrial-Specific Terms

| Term | Definition |
|------|------------|
| **Clear Height** | Interior ceiling height (18' is premium for industrial). |
| **Cross-Dock** | Facility where goods are unloaded and immediately reloaded. |
| **Drive-In Door** | Grade-level loading door (vs. dock-high). |
| **Dock-High** | Loading dock at truck bed height (48-52 inches). |
| **Flex Space** | Combination of warehouse and office space. |
| **Cold Shell** | Building delivered with structure only, no HVAC/finished spaces. |
| **Warm Shell** | Building with basic HVAC and some finishes, ready for TI. |
| **Tilt-Up Construction** | Concrete wall panels poured on-site and tilted into place. |

## Office-Specific Terms

| Term | Definition |
|------|------------|
| **Rentable vs. Usable** | Rentable includes load factor (common areas); usable is actual office. |
| **Load Factor** | Percentage markup from usable to rentable SF (typically 10-20%). |
| **Build-to-Suit** | Custom-built office for specific tenant. |
| **Spec Suite** | Pre-built office space ready for immediate occupancy. |
| **LEED Certification** | Green building certification (impacts rent premium). |

## Retail-Specific Terms

| Term | Definition |
|------|------------|
| **Anchor Tenant** | Major tenant that draws traffic (grocery, department store). |
| **In-line Space** | Retail units between anchors in shopping center. |
| **Outparcel** | Standalone pad site in shopping center parking lot. |
| **Co-Tenancy Clause** | Tenant can reduce rent if anchor tenant leaves. |
| **Percentage Rent** | Rent = base + % of gross sales above threshold. |
| **CAM Charges** | Common area maintenance (landscaping, parking lot, etc.). |

## Multifamily-Specific Terms

| Term | Definition |
|------|------------|
| **Unit Mix** | Distribution of studio/1BR/2BR/3BR units. |
| **Loss to Lease** | Difference between actual rent and market rent. |
| **Concessions** | Move-in specials (1 month free, reduced deposit, etc.). |
| **Economic Occupancy** | Actual rent collected / potential rent (accounts for vacancy + concessions). |
| **Physical Occupancy** | Percentage of units with signed leases. |
